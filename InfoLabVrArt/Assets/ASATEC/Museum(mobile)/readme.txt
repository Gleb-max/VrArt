[About the contents]
This data is a model that is supposed to be placed in the scene as a game object.
Prefab data is stored in the "01_Prefab" folder, so place it here.

The "02_Data" folder contains basic data such as FBX and materials.
If you want to make fine adjustments, please adjust this data.

Detailed appearance etc. are adjusted with "Unity.2019.4.28f1".
Set "Player Settings -> Other Settings -> Color Space" to [Linear]. If you do not change it to [Linear], the object may be too bright and white.
